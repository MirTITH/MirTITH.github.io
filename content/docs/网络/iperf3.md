> 官网：<https://iperf.fr/>

用于测量两台主机之间的网络通信速率，支持 TCP 和 UDP。

## 服务端

```bash
iperf3 -s -p 5201
```

参数说明：

- `-s`：服务端模式
- `-p`：监听端口（可省略，默认 `5201`）

## 客户端

### 测试 TCP 速度

```bash
iperf3 -c host -t 10 -p 5201 -R
```

参数说明：

- `-c`：客户端模式
- `host`：服务端 IP 或主机名
- `-t`：测试时长，默认 `10` 秒
- `-p`：服务端端口
- `-R`：测试接收速度；去掉后测试发送速度
- `-6`：使用 IPv6
- `-4`：使用 IPv4

### 测试 UDP 速度

```bash
iperf3 -u -c host -b 100m -t 10 -p 5201 -R
```

参数说明：

- `-u`：UDP 模式
- `-b`：目标带宽，默认 `1 Mbps`
- 其余参数与 TCP 模式相同
